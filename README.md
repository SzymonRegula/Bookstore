## How to run the project locally

1. Navigate to the project directory:

   ```
   cd Bookstore
   ```

2. Install dependencies:

   ```
   npm install
   ```

3. If necessary, install concurrently globally:

   ```
   npm install -g concurrently
   ```

4. Run the project:

   ```
   npm start
   ```
