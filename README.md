## How to run the project locally

1. Navigate to the project directory:

   ```
   cd Bookstore-main
   ```

2. Install dependencies:

   ```
   npm install
   npm install nodemon
   ```

3. If necessary, install concurrently:

   ```
   npm install concurrently
   ```

4. Run the project:

   ```
   npm start
   ```
